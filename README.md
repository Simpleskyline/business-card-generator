# Business Card Generator

A professional business card generator built with React and Tailwind CSS.
Fill in your details and get a live preview that updates as you type, then download your card as a high-resolution PNG.

## Live Demo

[View on Vercel](https://business-card-generator-three.vercel.app)

## Features

- Live preview that updates as you type
- Elegant serif design with warm tone palette
- Download as high-resolution PNG
- Fully responsive layout

## Built With

- React (useState, useRef)
- Tailwind CSS
- html2canvas

## What I Learned

- React component architecture
- Props and one-way data flow
- useState for live state management
- useRef for direct DOM access
- Conditional rendering with &&
